# QCODE 二维码、短链接 生成工具
一个 Chrome 拓展程序，生成基于当前页面的二维码和短网址

## 安装
[Chrome 网上商店](https://chrome.google.com/webstore/detail/qcode/aefjmcpolambjjegdnmpcblmldcoebdm/related?utm_source=InfinityNewtab)
